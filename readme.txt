== Description ==

IntroduceYou plugin allows you to introduce yourself without having to be registered on the wordpress blog. Talk about yourself, about some project you are supporting or anything you like.

The plugin is availaible on pages, and is displayed thanks to its widget.

== Installation ==

1. download zip file
2. upload zip file into to `/wp-content/plugins/` directory
3. Unzip file (you should have `/wp-content/plugins/introduce-you`)
4. Activate plugin 'Admin >> Plugins >> Installed' and click on activate under the "IntroduceYou" section.
5. Go to "Appearance >> Widgets" and move "Introduce You Widget" in any sibebar you want.
6. Enjoy.

== Frequently Asked Questions ==

1. The plugin is not showing, even if I activated it and the check box says it is "active".
>> Answer : Make sure the widget is active too ("Appearance >> Widgets", Cf. Installation section of this file)

2. I don't want to use the widget, how can I use the plugin anyway ?
>> Answer : You can use the plugin by calling function "introduceYouDisplay();" no parameters needed.
